MusePM: Microservice Enabling Package Manager

Use APIs Better!

Muse is a package manager for services. With muse, you can enable new services, install dependencies, manage API keys, and track usage. If you are paying for third party services to deliver application functionality or if you are starting a new project that will use API services, Muse can help.

Muse is currently in private beta.

Install Muse

Muse command line tools are written in Javascript, easily installable with npm. Once you have it installed, initialize your first project.

$ npm install -g musepm
$ mkdir ~/myproject; cd ~/myproject
$ musepm init
Created musepm.json, project type node.

Enable Services

With muse, setting up a new API service is as easy as installing a package. One command line invocation and you are done. Dependencies are automatically added based on your project type.

$ npm enable slack
Detected project type node. Installing slack
Slack Subdomain: example
Slack Login:
Slack Password:
Bost name: MuseBot
API key saved in musepm.json

Use the API

API keys and client libraries are automatically configured for use via the muse factory.

var musepm = require("musepm");
var bot = musepm.factory("slack", {bot: "MuseBot"});
bot.join("general");
bot.send("general", "Hello World");

Manage API Keys

API keys are automatically separated from code. They can be stored in the project, your home directory, or managed by a third party secret store. Production keys can easily be substituted at deployment time without complex scripting.

Track Usage

Muse easily instruments your API calls and reports usage to a central dashboard. Be immediately alerted to new applications, potentially security issues, and run away API usage. Know which parts of an API are being used, and by which applications.

Control Costs

With Muse, you can be alerted to spikes in costly API usage immediately, before an outsided monthly bill arrives. And you can also see when you are paying for more capacity than is needed, or using a more expensive service than is required. Muse is the ultimate tool for API vendor relationship management.
